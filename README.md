# Fredrik's dotfiles

This repo automates a fresh installation of a macOS exactly how Fredrik wants it.


### Install everything
```
mkdir ~/.dotfiles && cd ~/.dotfiles
git clone https://github.com/fsavje/dotfiles.git .
make
```
